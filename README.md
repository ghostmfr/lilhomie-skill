# lilhomie-skill

Clawdbot skill for [lilhomie](https://github.com/ghostmfr/lilhomie) - HomeKit REST API for macOS.

- Skill file: `SKILL.md`
- Homepage: https://github.com/ghostmfr/lilhomie

## Usage

Install this skill in your Clawdbot environment:

```bash
clawdhub install ghostmfr/lilhomie-skill
```

Or manually add to your skills directory.

## What it does

Teaches your AI assistant how to control HomeKit devices via the lilhomie REST API running on `localhost:8420`.

## Requirements

- macOS 13.0+
- [lilhomie.app](https://github.com/ghostmfr/lilhomie/releases) installed and running
- HomeKit devices configured
